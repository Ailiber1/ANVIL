## 2. デザインコンセプト

### 参考サイト3つ（各サイトから取り入れる要素）

| サイト | 取り入れる要素 |
|--------|--------------|
| **comme.fit** | セリフ体英字の大きなあしらい / 余白の贅沢な使い方 / スムーズなスクロール体験 |
| **BEYOND** | モノトーン+1色アクセントの配色 / 左右交互の画像・テキスト配置 / 4ステップフロー図 / CTA配置の巧みさ |
| **Puretec Fitness** | ダーク背景×ゴールドのプレミアム感 / フルスクリーンヒーロー画像 / 大胆なタイポグラフィ |

### Tone
**powerful-refinement（力強さの中にある洗練）**
— ゴリゴリの筋肉推しではなく、「ブランドとしてのパーソナルトレーニング」を表現。
comme.fitのブランド感 × Puretecのダークプレミアム感を融合。
男女問わず「かっこいい」と感じるトーン。

### Differentiation
「パーソナルジムのサイトなのに、ファッションブランドのような世界観がある」

### 5軸デザイン判断

| 軸 | 判断 | 理由 |
|---|---|---|
| **Typography** | Cormorant Garamond（英字見出し・セクション番号・装飾テキスト）+ Noto Sans JP（日本語見出し・本文） | comme.fit式のセリフ体英字あしらいで高級感。日本語はゴシックで可読性確保。英字を大きく飾りとして使うのが今風 |
| **Color** | 背景: Obsidian `#0A0A0A` / テキスト: Off White `#F5F5F0` / アクセント: Muted Gold `#C9A96E` / サブ背景: Dark Charcoal `#141414` / サブテキスト: Warm Gray `#9A9A9A` | Puretec式の黒+ゴールド。ゴールドは彩度を落とした「くすみゴールド」でギラつかせない。comme.fitのニュアンスカラー感を黒ベースで再解釈 |
| **Motion** | fade-in-up（opacity + translateY, 0.7s ease）/ 数字カウントアップ（実績セクション）/ ヘッダーのスクロール変化 / 画像のスケールイン（scale 1.05→1.0） | 派手すぎないが「動いてる」と感じる程度。comme.fitの上品さを参考。パララックスは使わない（重くなる＋LPには過剰） |
| **Spatial** | セクション間: 120px / カード内padding: 40px / テキスト行間: 1.8〜2.0 / 最大幅: 1200px | comme.fit式の広い余白。「余裕＝高級」を徹底。情報を詰め込まない |
| **Texture** | ダーク背景にnanobanana proで生成した写真を部分的に配置。写真にはダークオーバーレイ（black/40〜60%）をかけてテキストの可読性を確保。セクション間の切り替えはソリッドカラーの交互使用（#0A0A0A ↔ #141414） | 写真が映える暗い背景。オーバーレイで統一感を出す（Fenriz Gym式のフォトフィルター統一手法） |

### AIスロップ排除ルール
- 紫グラデーション禁止
- `rounded-2xl`以上の角丸禁止（`rounded-lg` = 8pxまで）
- 3カラム均一カードレイアウトは極力避ける（使う場合はカードサイズに変化をつける）
- 「Welcome to...」「ようこそ」系のヒーロー禁止
- 意味のないアイコン羅列禁止
- グラデーションボタン禁止（ソリッドカラーのみ）
- パーティクル・星空・浮遊要素禁止
- ムキムキのストック写真・ベタなトレーニング写真禁止（nanobanana proで雰囲気重視の画像を生成）
- 赤・オレンジ等の「エネルギッシュ」配色禁止（ジムサイトあるある。今回はゴールド×黒の洗練路線）
- 「あなたの理想の体を手に入れよう！」的な安易なコピー禁止

### デザインの適用範囲
上記のダークテーマはトップページ（LP部分）に適用。
予約フォーム・管理ダッシュボードはダークテーマを維持しつつ、操作性を優先した設計にする（入力フォームは背景を少し明るく `#1A1A1A` にする等）。

- `invoice.payment_succeeded` → 月額課金成功の記録
- `invoice.payment_failed` → 管理者に通知
- `customer.subscription.deleted` → 退会処理

※ デモサイトのためStripeはテストモードで運用
